# Counter Web Application

A simple and interactive Counter Web Application built using HTML, CSS, and JavaScript. This project allows users to increase or decrease a counter value and visually indicates whether the value is positive or negative.

## 📌 Features
- Increase and decrease counter value
- Dynamic update of counter display
- Visual feedback:
  - Green border for positive values
  - Red border for negative values
- Clean and simple user interface

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla JS)

## 📂 Project Structure

├── index.html
├── style.css
├── script.js
